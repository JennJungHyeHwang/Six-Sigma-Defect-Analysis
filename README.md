# Six-Sigma-Defect-Analysis
Six Sigma analysis of vehicle defects using NHTSA data in R

<-- (project in link)

This project analyzes vehicle defect data obtained from the National Highway Traffic Safety Administration (NHTSA), focusing on common issues such as transmission downshift, timing belt failure, and B-pillar trim defects. Using Six Sigma methodology, the defect rate and sigma level were calculated to evaluate overall quality performance. A simulation was then conducted to assess how reducing defects in the most prevalent category (transmission downshift) impacts overall process performance. The results show that improvements in the primary defect can significantly increase the sigma level and reduce overall defect rates.
